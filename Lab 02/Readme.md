# Lab 02 - Introduction to Data Analysis Libraries

🚀 **Machine Learning Lab 02 | NumPy, Pandas & Matplotlib Fundamentals**

This lab introduces the basic Python libraries commonly used in Machine Learning and Data Science. The lab focuses on **NumPy** for numerical computing, **Pandas** for data handling, and **Matplotlib** for data visualization.

## 📖 Lab Overview

Lab 02 provides the foundation required for future machine learning labs. Before building models, it is important to understand how to create arrays, handle tabular data, inspect datasets, and visualize values using plots.

The lab is divided into three notebooks:

- `Lab_2.1_22jzele0467.ipynb` - NumPy fundamentals
- `Lab_2.2_22jzele0467.ipynb` - Pandas data manipulation
- `Lab_2.3_22jzele0467.ipynb` - Matplotlib data visualization

## 🎯 Learning Objectives

By completing this lab, students will be able to:

- ✔ Import and use NumPy, Pandas, and Matplotlib
- ✔ Create arrays using NumPy
- ✔ Understand array indexing, slicing, shape, and matrix operations
- ✔ Create and manipulate Pandas Series and DataFrames
- ✔ Read and inspect tabular data
- ✔ Perform basic data selection and filtering
- ✔ Create simple plots using Matplotlib
- ✔ Add plot titles, labels, grids, markers, and colors
- ✔ Understand the role of data analysis libraries in machine learning

## 📚 Notebook Details

### 🔹 Lab 2.1 - Introduction to NumPy

This notebook introduces NumPy, which is used for numerical computation in Python.

Topics covered:

- Importing NumPy
- Creating arrays from Python lists
- One-dimensional and multi-dimensional arrays
- Array shape and dimensions
- Indexing and slicing
- Matrix creation
- Random arrays
- Basic numerical operations

NumPy is important in machine learning because most datasets and model inputs are represented in numerical array form.

### 🔹 Lab 2.2 - Introduction to Pandas

This notebook introduces Pandas for tabular data handling and analysis.

Topics covered:

- Importing Pandas
- Creating DataFrames from dictionaries
- Creating Series
- Reading CSV files
- Displaying rows using `head()` and `tail()`
- Checking data information using `info()`
- Summary statistics using `describe()`
- Data selection and indexing
- Grouping and filtering operations

Pandas is widely used in machine learning for dataset loading, cleaning, inspection, and preprocessing.

### 🔹 Lab 2.3 - Introduction to Matplotlib

This notebook introduces Matplotlib for plotting and data visualization.

Topics covered:

- Importing Matplotlib
- Creating line plots
- Adding titles and axis labels
- Adding grids and legends
- Using markers and colors
- Bar charts
- Scatter plots
- Histograms
- Subplots

Visualization helps in understanding data trends, distributions, and relationships before applying machine learning models.
