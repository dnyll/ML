# Machine-Learning-Labs

🚀 **Machine Learning & Deep Learning Lab Portfolio**

A comprehensive collection of Machine Learning and Deep Learning laboratory exercises, practical implementations, and experiments developed throughout the **Machine Learning Lab** course. This repository demonstrates hands-on application of modern AI techniques using industry-standard tools and frameworks, covering topics from data preprocessing and machine learning algorithms to advanced deep learning architectures.

## 📖 About the Repository

This repository serves as a practical learning resource for students, researchers, and AI enthusiasts interested in Machine Learning and Deep Learning.

The labs are designed to provide hands-on experience with:

- Data analysis and visualization
- Data cleaning and preprocessing
- Feature engineering
- Machine learning algorithms
- Deep learning architectures
- Computer vision applications
- Time-series forecasting
- Neural network design and optimization
- CityLearn demand forecasting

Each laboratory module focuses on a specific concept and includes implementation, experimentation, visualization, and performance evaluation.

## 🎯 Learning Objectives

By completing these laboratory exercises, students will be able to:

- ✔ Perform exploratory data analysis (EDA)
- ✔ Clean and preprocess real-world datasets
- ✔ Apply feature engineering techniques
- ✔ Build and evaluate machine learning models
- ✔ Design and train deep learning architectures
- ✔ Develop computer vision solutions using CNNs
- ✔ Analyze and forecast sequential data
- ✔ Implement AI solutions using TensorFlow and Keras
- ✔ Work with time-series forecasting models such as MLP, 1D CNN, and LSTM

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Machine-Learning-Labs.git
cd Machine-Learning-Labs
```

Install the required dependencies:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow keras opencv-python citylearn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
